TEMPLATES1 – Classic2 – Light3 – Black Pearl4 – Xillia5 – Little Garden6 – Six Party7 – Arcana8 – Eternal Sonata9 – Marshmallow10 – Classic II11 – Planetariun12 – Honey Bee
CARACTERÍSTICAS
O plugin permite fazer customizações avançadas no layout de batalha .
UTILIZAÇÃO
As imagens devem ser gravadas na pasta /img/battlehud/ 

Basicamente para configurar o plugin basta mudar a posição X e Y das imagens e janelas para deixar o layout da batalha da maneira que desejar.
ANIMATED FACES
É possível ativar faces animadas através da opção. (Face Frame Animation)Exemplo de face Animada.
TEMPLATES
Quando for substituir os arquivos dos TEMPLATES.
Certifique de primeiro desistalar o Plugin do MOG_BattleHud e após isso salvar, isso é necessário para que o programa do Rpg Maker zere todas as configurações feitas pelo plugin anterior, caso contrário o setup ficará com os valores antigos e você terá o resultado abaixo.

CUSTOM POSITION
Para definir uma posição especifica da Hud , vá até opção CUSTOM POSITION e defnina um valor X e Y para cada Hud.
Essa função ignora o setup do Hud X-axis e Hud Y-axis.
PLUGIN COMMANDS
Para ocultar a hud use o comando abaixo.bhud_disable
Para apresentar a hud use o comando abaixo.bhud_enable
SCRIPT COMMAND
Para ocultar ou apresentar a hud use comando abaixo.$gameSystem._bhud_visible = false
NOTETAGS
Para ativar o efeito de respirar nas faces utilize a notetag abaixo.
Face Breath Effect
