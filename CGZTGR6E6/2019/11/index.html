lsp 系のプラグインであれば、tsconfig.json の include に指定されている glob に含まれるモジュールは、補完候補に出てきます。（そして、その補完候補を選択すると自動でインポートされることがあります。これはプラグインの対応状況次第。）

また、lsp 系のプラグインであれば、CodeAction から import を挿入できます。
（とりあえず、インポートしてないモジュールを使うコードをガガガっと書いてから、CodeAction でインポートしていく。みたいなやり方ができます。）
