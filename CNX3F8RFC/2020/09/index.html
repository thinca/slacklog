     set-clipboard [on | external | off]
             Attempt to set the terminal clipboard content using the xterm(1) escape sequence, if there is an Ms entry in the terminfo(5) description (see the TERMINFO
             EXTENSIONS section).

             If set to on, tmux will both accept the escape sequence to create a buffer and attempt to set the terminal clipboard.  If set to external, tmux will attempt
             to set the terminal clipboard but ignore attempts by applications to set tmux buffers.  If off, tmux will neither accept the clipboard escape sequence nor
             attempt to set the clipboard.

             Note that this feature needs to be enabled in xterm(1) by setting the resource:

                   disallowedWindowOps: 20,21,SetXprop

             Or changing this property from the xterm(1) interactive menu when required.
