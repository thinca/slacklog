や

" in Xor128.vim
function! s:srand(...) abort
  if a:0 == 0
    let x = has('reltime') ? reltime()[1] : localtime()
  elseif a:0 == 1
    let x = a:1
  else
    throw 'vital: Random.Xor128: srand(): too many arguments'
  endif
  call s:_common_generator().seed([x])
endfunction

なので、srandは数値を受けるのに ([{seed}]) と記載して、 seedはリストを受けるのに ({seeds}) になってるな、と
