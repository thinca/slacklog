slacklog が公開されたので、削除の際のオペレーションを決めたいと思います。基本、権限が無いと消せないので管理者にエスカレーションして下さい。
日時バッチなので、その日の夜までに消せばアーカイブに乗らないので問題ないですが、それを超えると GitHub Pages に公開されてしまうので GitHub 社にお願いしてコミットを消さないと一生インターネットに公開されてしまう事になります。この辺も、ログを公開した責任として管理者がやる事になります。
そうなった場合、まずは GitHub Pages からの削除、必要であれば GitHub 社にお願いしてコミットの削除までやる事になります。
