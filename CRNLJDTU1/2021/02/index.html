【業務連絡】slacklog について

slacklog の JSON は更新されているものの、html ページが生成にコケ続けているので、応急措置として私の私用マシンで毎朝5時に html を生成して push するようにスクリプトを仕込みました。 (実際に動くのは今晩が初なのでうまく動くかはまだわからない)
GitHub Actions の html 生成 workflow は、手動で無効化しました。
生成の過程でバグをみつけたので修正 PR を投げてあります。 https://github.com/vim-jp/slacklog-generator/pull/155 (ローカル環境では適用済み)
