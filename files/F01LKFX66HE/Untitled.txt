## Procedure

```go
package main

import "fmt"

func main() {
	fmt.Println(|fmt.Sprintf("%s", "foo"))
}

1. Move the cursor to the `|` position to enter insert mode. 
1. Type `fmt.Spri`
1. `C-X, C-O`(Activate omni-completion) and select `Sprintf`
1. Type `(`
```

## Micro vimrc

```vim
call plug#begin('~/.local/share/nvim/plugged')

Plug 'prabirshrestha/vim-lsp'

call plug#end()

if executable('gopls')
    augroup LspGo
        autocmd!
        autocmd User lsp_setup call lsp#register_server({
          \ 'name': 'gopls',
          \ 'cmd': {server_info->['gopls', '-mode', 'stdio']},
          \ 'whitelist': ['go'],
          \ })
    augroup END
endif

set omnifunc=lsp#complete
```
