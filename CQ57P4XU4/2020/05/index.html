vim-efm-langserver-settings は eslint にファイル名を渡しているので、
efm-langserver の README.md での設定例との差が出ているのだと思っています。
(.eslintrc.js とかを特別な設定なしで読むと再現すると思います)

( zero-based が 0以上 であることを要求してるのか それとも type が number だから OK なのかちょっと判断できませんでした )

vim-efm-langserver-settings の設定はこのままでいいと思っています
