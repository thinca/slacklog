                        *expr-is?* *expr-isnot?*
                'ignorecase'次第   大小文字考慮    大小文字無視  
等しい                  ==              ==#             ==?
等しくない              !=              !=#             !=?
より大きい              >               >#              >?
より大きいか等しい      >=              >=#             >=?
より小さい              <               <#              <?
より小さいか等しい      <=              <=#             <=?
正規表現マッチ          =~              =~#             =~?
正規表現非マッチ        !~              !~#             !~?
同一のインスタンス      is              is#             is?
異なるインスタンス      isnot           isnot#          isnot?
 
例:
"abc" ==# "Abc"   0と評価される
"abc" ==? "Abc"   1と評価される
"abc" == "Abc"    'ignorecase' が設定されていれば1と、でなければ0と評価
