端末によって設定を切り替える　　　　　　　　　　　　*term-dependent-settings*
　
端末名によってオプションやマッピングを切り替えるには .vimrc で行うのがベスト
です。例: >
　
if &term == "xterm"
... xterm 用のマッピングと設定 ...
elseif &term =~ "vt10."
... vt100, vt102 用のマッピングと設定 ...
endif
<
