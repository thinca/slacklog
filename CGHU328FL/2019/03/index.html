# イベント内容
girls.vim vol.1

# 概要
vimを使いこなせるようになりたい！vimが好き！そんな女性エンジニア向けのイベントの第一弾！
今回はvimを使いこなせるようになろうということをテーマにLT&もくもく会という盛りだくさんな内容になります。
第一回目ということで、参加者は女性の方に限らせていただきます。

# 開催日

# 持ち物
ノートPC
↑もくもく会でvimのインストールをサポートできるかどうかで事前にvimが必要かどうかかわる

# 対象
- vimをもう少し使えるようになりたい方
- vimが好きな方
- 人がvim触ってるところをみていたい方
- vimのプラグインをつくってみたい方
- 女性エンジニア仲間を増やしたい方

# 定員
10名？会場的にもう少し多くても良さそうだけれどそんなあつまる？
LT枠3名、スタッフ枠3名?、一般枠10くらい？

# 会場
Quipper 目黒オフィス
東京都品川区上大崎2-13-30 oak meguro 5F

# タイムテーブル
